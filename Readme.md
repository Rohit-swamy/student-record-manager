# Student Record Manager (Python)

## Overview

The Student Record Manager is a command-line Python application that allows users to manage student records using file handling. It provides an easy way to add, search, view, and analyze student information while storing the data permanently in a text file.

This project was built as part of my Python learning journey to practice core programming concepts and improve problem-solving skills.

## Features

* Add new student records
* View all student records
* Search a student by name
* Count the total number of students
* Find student(s) with the highest marks
* Find student(s) with the lowest marks
* Permanent data storage using file handling
* Input validation using exception handling
* Handles missing files gracefully
* Handles empty file scenarios
* User-friendly menu-driven interface
